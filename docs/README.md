基础功能
=======

- 账号中心
  - [x] 门户：背景图轮播
  - [x] 账号：注册、登录
  - [ ] 剧本：不同的剧本属于不同的游戏内容，当然也可以是同一个剧本多个舞台演绎（游戏多区）
- 后台管理
    - [ ] 初始化：管理后台的管理员账号
    - [ ] 管理后台（持续完善）
- 核心服务
    - [ ] 舞台：展示剧本包含的内容
    - [ ] 职业：一转（战士、魔法师、弓箭手），二转（未完待续）
    - [ ] 地图：世界地图、副本地图、活动地图等等
    - [ ] 剧情：周期主题、一次性主线和支线、日常循环
    - [ ] 玩家：创建游戏角色、选择职业、进入地图、开始剧情
    - [ ] 道具：除装备之外的各种消耗品或任务用品
    - [ ] 装备：装饰、服饰、武器、防具等等
    - [ ] 放置：解放双手、按期签到即可免费享用
